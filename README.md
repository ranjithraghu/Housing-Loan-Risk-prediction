+ACM-  Housing Loan Prediction Project

+ACMAIw-  Project Overview
This project focuses on predicting whether a customer is eligible for a housing loan using Machine Learning techniques. The main objective is to help banks and financial institutions automate loan approval decisions based on customer financial history and repayment behavior.

The project includes:
- Data Cleaning +ACY- Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Building
- Model Evaluation +ACY- Comparison

---

+ACM-  Problem Statement
Financial institutions face major challenges in identifying customers who may default on loans. Incorrect approvals can increase financial risk and losses.

This project aims to build a predictive machine learning model to:
- Predict loan repayment capability
- Reduce loan default risk
- Improve approval efficiency
- Support data-driven financial decisions
